# MixinBot

An API wrapper for [Mixin Network](https://developers.mixin.one/api)

> Notice: this project is ongoing.

## Installation

Add to gemfile, and `bundle install`

```
gem 'mixin_bot', github: 'an-lee/mixin_bot', branch: 'master'
```
